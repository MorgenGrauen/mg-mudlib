P_TRAY
======

NAME
----

  P_TRAY                        "tray"                        

DEFINIERT IN
------------

  /sys/properties.h

BESCHREIBUNG
------------

  Markiert Objekte als Tray. Derzeit hat dies nur den Effekt, das Spieler
  Objekte darauf ablegen und sie herunternehmen koennen.

ANMERKUNGEN
-----------

  Bei von /std/tray geerbten Objekten wird die Property automatisch gesetzt.
