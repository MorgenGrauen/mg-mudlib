P_LOCATION
==========

NAME
----
::

    P_LOCATION                   "location"

DEFINIERT IN
------------
::

    /sys/player/base.h

BESCHREIBUNG
------------
::

    Hier kann der Spieler mit dem Befehl "ort" den Ort setzen,
    von dem er kommt bzw. zu kommen glaubt ;)
    Um wieder den automatisch ermittelten Ort anzuzeigen, ist
    P_LOCATION auf 0 zu setzen.

    

SIEHE AUCH
----------
::

    ort
