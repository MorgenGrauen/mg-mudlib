P_NOBUY
=======

NAME
----
::

    P_NOBUY                       "nobuy"                       

DEFINIERT IN
------------
::

    /sys/properties.h

BESCHREIBUNG
------------
::

     Wenn diese Property gesetzt ist, wird das Objekt nach einem
     Verkauf im Laden zerstoert, damit es nicht wieder von einem Spieler
     gekauft werden kann.
