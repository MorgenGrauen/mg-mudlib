P_FOOD_DELAY
============

NAME
----
::

    P_FOOD_DELAY                 "food_delay"                     

DEFINIERT IN
------------
::

    /sys/living/life.h

BESCHREIBUNG
------------
::

     Anzahl der heart_beats, bis P_FOOD um einen Punkt sinkt.
     Aenderungen dieser Property in Spielern beduerfen der 
     Genehmigung des EM fuer Balance.
