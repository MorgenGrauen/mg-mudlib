P_ALCOHOL
=========

NAME
----
::

     P_ALCOHOL			"alcohol"

DEFINIERT IN
------------
::

     /sys/living/life.h

BESCHREIBUNG
------------
::

     - Lebewesen
       Numerischer Wert fuer den Grad der Betrunkenheit eines Lebewesens.
       Der maximale Wert steht in P_MAX_ALCOHOL.

     - Speisen/Kneipen
       Numerischer Wert fuer den Grad, den eine Portion der Speise den
       Konsumenten betrunken macht
