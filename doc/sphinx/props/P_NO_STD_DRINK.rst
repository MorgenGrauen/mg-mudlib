P_NO_STD_DRINK
==============

NAME
----
::

	P_NO_STD_DRINK			"no_std_drink"

DEFINIERT IN
------------
::

	/sys/pub.h

BESCHREIBUNG
------------
::

        Durch setzen dieser Property in einer Kneipe sorgt man dafuer, dass 
        "Standard-Drinks" (z.B. Gluehwein im Dezember) nicht in das Menue
        der Kneipe aufgenommen werden.

BEMERKUNGEN
-----------
::

        Keine.
