P_TIMEZONE
==========

NAME
----
::

    P_TIMEZONE                 "timezone"

DEFINIERT IN
------------
::

    /sys/player/base.h

BESCHREIBUNG
------------
::

     Ein Integer-Wert, der bei der Uhrzeitmeldung und beim Befehl
     "(uhr)zeit" beruecksichtig wird. Gibt die Anzahl der Stunden
     Zeitabweichung von Berliner Zeit an.
