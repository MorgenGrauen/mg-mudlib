P_PREPARED_SPELL
================

NAME
----

  P_PREPARED_SPELL  "prepared_spell"

DEFINIERT IN
------------

  /sys/new_skills.h

BESCHREIBUNG
------------

  Enthaelt ein Array mit Daten zu einem in Vorbereitung befindlichen Spell.

  - arr[0]: Ausloesezeitpunkt (int)
  - arr[1]: Spellinformationen (mapping)
  - arr[2]: Skill-Argument (string)
