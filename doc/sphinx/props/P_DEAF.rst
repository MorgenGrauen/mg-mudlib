P_DEAF
======

NAME
----
::

    P_DEAF                        "deaf"                        

DEFINIERT IN
------------
::

    /sys/player/comm.h

BESCHREIBUNG
------------
::

     Der Spieler ist taub. Falls hier ein String steht, wird dieser bei
     "teile ... mit" an den Mitteilenden ausgegeben, ansonsten kommt nur
     "Soundso ist leider gerade taub.\n"
