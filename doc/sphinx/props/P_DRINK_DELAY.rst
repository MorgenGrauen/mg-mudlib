P_DRINK_DELAY
=============

NAME
----
::

    P_DRINK_DELAY                 "drink_delay"                     

DEFINIERT IN
------------
::

    /sys/living/life.h

BESCHREIBUNG
------------
::

     Anzahl der heart_beats, bis P_DRINK um einen Punkt sinkt.
     Aenderungen dieser Property in Spielern beduerfen der 
     Genehmigung des EM fuer Balance.
