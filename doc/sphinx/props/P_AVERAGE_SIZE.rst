P_AVERAGE_SIZE
==============

NAME
----
::

    P_AVERAGE_SIZE                "average_size"                

DEFINIERT IN
------------
::

    /sys/player/description.h

BESCHREIBUNG
------------
::

   Durchschnittliche Groesse eines Wesens dieser Rasse (in cm).
   Bei den Rassen im Morgengrauen sind das:

		Menschen    175
		Magier      sind im Durchschnitt etwas groesser ;-)
		Orks        195
		Hobbits     105
		Goblins      80
		Felinen     200
		Zwerge      120
		Elfen       195
		Dunkelfen   175
