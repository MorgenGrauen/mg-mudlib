P_SHOW_EXITS
============

NAME
----
::

    P_SHOW_EXITS                  "show_exits"

DEFINIERT IN
------------
::

    /sys/player/base.h

BESCHREIBUNG
------------
::

     Im Spieler gesetzt, wenn der Spieler die offensichtlichen Ausgaenge
     immer automatisch sehen will.

SIEHE AUCH
----------
::

     Aehnliches:	P_HIDE_EXITS
     Sonstiges:		AddExit(), GetExits(), int_long(), int_short()
