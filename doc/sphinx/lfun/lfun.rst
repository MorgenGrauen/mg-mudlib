lfun()
======

NAME
----
::

	lfun()

DESCRIPTION
-----------
::

	This directory contains descriptions for the lfuns used by
	Amylaar's version of the LPC parser.

	These are functions that are applied by the parser to the LPC
	objects on various occasions.
